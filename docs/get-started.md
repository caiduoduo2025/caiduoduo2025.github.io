# 快速入门

欢迎使用 **财多多** —— 你的智能投资理财管家。

请在电脑上按照以下步骤快速开始使用：

## 1. 环境准备

请前往 [Oracle 官网](https://www.oracle.com/java/technologies/downloads/) 下载并安装 **Java 21** 或更高版本。

## 2. 下载软件

点击下方链接下载最新版本财多多软件：

- [下载 cdd-1.0.jar](/download.html?v=1.0)

## 3. 启动软件

打开终端（Terminal）或命令行工具，执行以下命令启动软件（有问题可求助各种AI助手）：

```bash
java -jar cdd-1.0.jar
```

启动后，在你的浏览器中打开 [http://localhost:8888/](http://localhost:8888/) 你将看到软件的激活页面。

## 4. 获取激活码

请添加客服微信申请激活码：

- **微信号**：`caiduoduokefu`
- **申请备注**：请务必备注你的**邮箱地址**（用于接收激活码）。

## 5. 完成激活

关注你的邮箱，请在收到激活码 **72小时内** 在软件中完成激活。

## 6. 开始使用

激活成功后，你可以立即开始：

- **创建组合**：建立你的第一个投资组合。
- **入金记录**：记录你的初始资金。
- **添加资产**：添加你持有的股票、基金或其他资产。
- **交易记录**：记录买卖操作。
- **组合分析**：查看收益曲线和资产分布。

祝你使用愉快，财源滚滚！
