# 价格更新配置

本系统支持通过配置外部 API 来自动更新资产价格。以下是关于配置 URL 和 Curl 命令的详细说明。

## URL 占位符替换

系统会自动识别并替换 URL 中的具体股票代码为 `{SYMBOL}` 占位符。这意味着你不需要手动修改 URL 中的股票代码，系统会根据当前查询的资产自动填充。

## Curl 命令支持

系统支持直接粘贴包含具体代码的 Curl 命令，简化配置过程。

### 自动替换

当你粘贴一个包含具体股票代码（如 `symbol=SH510300`）的 Curl 命令时，系统会自动将其变为 `symbol={SYMBOL}`。

### 格式支持

- **单行命令**：支持标准的单行 Curl 命令。
- **多行命令**：支持使用反斜杠 (`\`) 换行的多行 Curl 命令格式，方便阅读和复制。

### 注意事项

- 请确保 Curl 命令格式正确。
- 命令中必须包含必要的请求头（Headers）和参数，以确保请求能被目标服务器正确处理。

## 生效时间

配置更新后将**立即生效**，并影响后续所有的价格查询操作。

## 常见问题

### Curl 模板过期

某些数据源（如雪球）的 Curl 命令可能会因为 Cookie 或 Session 过期而失效。

**解决方法**：
1. 访问雪球首页。
2. 重新访问相关接口。
3. 拷贝新的 Curl 命令并更新到系统配置中。
